# Demo Away Team Animation Frame Render

This project use [Kaitai Struct](https://kaitai.io/) to parse the animation zip data from the game "Away Team Playtest" and adapt [`kanimtool`](https://github.com/tsangwpx/kanimtool) library to render the frames.

This project is meant to create animation files used for community game wiki. It's currently broken and frame elements are not rendered in the correct position, but it can be used as a reference for anyone who want to work on this.

Note this project doesn't contain any asset files.
